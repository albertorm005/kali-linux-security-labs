# Kali Linux Security Labs

This repository contains practical cybersecurity labs performed using Kali Linux.

The goal of this project is to document hands-on security testing techniques in a controlled lab environment.

## Topics Covered

- Network scanning with Nmap
- Vulnerability scanning with Nikto
- Brute force attacks with Hydra
- Exploitation using Metasploit

## Lab Structure

network-scanning  
vulnerability-scanning  
brute-force  
exploitation  

## Disclaimer

All tests are performed in a controlled lab environment for educational purposes only.
